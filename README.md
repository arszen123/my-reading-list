# MyReadingList

Reading list tracking application.

## Features

- Authentication
- Search books
- Save books (Set book status)
- View saved books
- Change book status

## Technologies

- React (Typescript)
- Firebase
    - Authentication
    - Realtime Database
    - Hosting
- Google Books API
- Chakra UI

## Scripts

```bash
# start local development server
npm start

# build the application
npm build

# lint the project
npm lint
```
